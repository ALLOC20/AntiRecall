# AntiRecall（QQ,Wechat防撤回）
[English](https://github.com/plummm/AntiRecall/blob/master/README-eng.md)

这款软件用于Win PC平台，针对QQ,Wechat撤回功能.

内存补丁模式更为稳定和高效

## 更新 6/1/2020

## 新功能

- 新增内存补丁模式，防撤回更加高效稳定
- 重写了ui
